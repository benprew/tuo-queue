# tou queue

A queuing API for running tuo sims

## Requirements
* Ruby
* [tuo](https://sf.net/p/tyrant-unleashed-optimizer)

## Installation

After cloning the repo run:

1. `gem install bundler`
2. `bundle install`

## Starting the queue

In the root directory of the repo run:

1. `bundle exec bin/tuo_queue start`
